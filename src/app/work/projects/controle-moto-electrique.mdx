## Description

Maquette UI/UX réalisée sur Figma pour une **application mobile de contrôle de moto électrique**.
L’application permet de gérer différents aspects de la moto via une interface intuitive et moderne.

## Fonctionnalités Conçues

- Dashboard principal avec indicateurs clés
- Interface de verrouillage/déverrouillage
- Monitoring de la batterie en temps réel
- Géolocalisation et historique des trajets
- Paramètres de configuration de la moto

## Compétences Démontrées

- Design mobile first
- Visualisation de données en temps réel
- Expérience utilisateur pour la mobilité
- Prototypage interactif

## Technologie Utilisée

- Figma

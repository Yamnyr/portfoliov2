## Description

Hurl Ia est un bot Discord conçu pour automatiser la gestion des comptes utilisateurs en fonction de leur rôle, tout en permettant l'exécution de requêtes directes dans une base de données (BDD) via le serveur Discord. Destiné aux équipes ayant besoin d'un accès rapide et sécurisé aux informations stockées dans une BDD, ce bot facilite la création de comptes et la gestion de rôles.

## Fonctionnalités

### Pour les utilisateurs

- **Gestion de comptes :**
  - Création de comptes en fonction du rôle attribué sur Discord
  - Connexion sécurisée et authentification via Discord

- **Exécution de requêtes en base de données :**
  - Envoi de requêtes de lecture et d'écriture dans la BDD directement via des commandes Discord
  - Gestion de permissions pour limiter l'accès à certaines commandes selon le rôle
  - Visualisation des données en temps réel


## Technologies utilisées

- Python, discord.py
- MySQL

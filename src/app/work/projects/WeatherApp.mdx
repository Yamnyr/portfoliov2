## Description

WeatherApp est une application météorologique moderne développée avec Angular. Elle permet aux utilisateurs d'accéder aux conditions météorologiques en temps réel via la géolocalisation et de consulter les prévisions pour n'importe quelle ville dans le monde grâce à un système de recherche intuitif.

## Fonctionnalités

### Météo en Temps Réel

- Détection automatique de la position de l'utilisateur
- Barre de recherche pour choisir une ville
- Affichage des données météorologiques actuelles :
     - Température
     - Ressenti
     - Humidité
     - Vitesse du vent

### Prévisions Météorologiques

- Prévisions horaires sur 24h
- Affichage des tendances :
     - Températures min/max
     - Type de temps (ensoleillé, nuageux, pluvieux)

### Interface Utilisateur

- Design responsive et moderne
- Animations fluides
- Thème dynamique basé sur :
    - Les conditions météo

## Technologies Utilisées

### Frontend
- Angular 16+
- TypeScript
- SCSS
- Angular Material

### APIs et Services
- OpenWeatherMap API
- Geocoding API
- HTML5 Geolocation API
- Progressive Web App (PWA)

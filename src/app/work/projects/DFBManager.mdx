## Description

DFBManager est un outil de gestion de projets développé sous Symfony, conçu pour permettre un suivi efficace des projets et des relations clients. L'application permet de centraliser les informations des projets, de suivre leur état d'avancement et de gérer les données clients de manière structurée.

## Fonctionnalités

### Gestion des Projets

- Création et édition de projets :
     - Informations générales
     - État d'avancement
     - Dates importantes
     - Budget et facturation
     - Documents associés
- Notes et commentaires internes

## Technologies Utilisées

### Backend
- Symfony 6.4
- PHP 8.1+
- Doctrine ORM
- MySQL
- Twig
- JavaScript
- Bootstrap 5

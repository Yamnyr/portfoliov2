## Description

Maquette UI/UX réalisée sur Figma pour une **application météo adaptée aux montres connectées**.
L’objectif était de concevoir une interface claire malgré les contraintes d’un petit écran.

## Fonctionnalités Conçues

- Design adapté aux contraintes d'écran
- Affichage des données météorologiques essentielles
- Navigation tactile intuitive

## Compétences Démontrées

- Design pour écrans réduits (smartwatch)
- Minimalisme et lisibilité
- Ergonomie tactile
- Tests utilisateurs

## Technologie Utilisée

- Figma

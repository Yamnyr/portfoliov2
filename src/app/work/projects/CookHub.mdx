## Description

CookHub est une plateforme communautaire dédiée au partage de recettes de cuisine. Elle permet aux utilisateurs de découvrir, créer et partager leurs recettes préférées tout en interagissant avec une communauté de passionnés de cuisine.

## Fonctionnalités


- Création et édition de recettes :
     - Titre, description et photo de présentation
     - Liste détaillée des ingrédients avec quantités
     - Étapes de préparation
     - Tags et catégories

- Gestion des recettes :
     - Modification de ses propres recettes
     - Suppression avec confirmation
     - Visualisation du statut (publié/brouillon)


- Recherche avancée

- Ajout/retrait de recettes aux favoris

- Notes et commentaires :
    - Notation sur 5 étoiles
    - Commentaires
    - modifications
    - Inscription/connexion

- Tableau de bord :
     - Ses recettes publiées
     - Ses recettes favorites
     - Historique d'activité

## Technologies utilisées

- React 18
- React Router pour la navigation
- Material-UI pour les composants
- Axios pour les requêtes API

- Node.js avec Express
- mysql
- expressjs
- JWT pour l'authentification

- Docker pour la containerisation
- GitHub Actions pour le CI/CD
- AWS S3 pour le stockage des images
- Nginx comme serveur proxy

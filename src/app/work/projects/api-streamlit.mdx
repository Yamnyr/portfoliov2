## Description

Ce projet se compose de deux applications distinctes utilisant des APIs publiques :
1. Une application de gestion de films utilisant l'API TMDB
2. Un simulateur de combats Pokémon utilisant l'API PokéAPI

## Fonctionnalités

### Application Films (TMDB)

  - Recherche de films par titre
  - Affichage des détails complets des films
  - Visualisation des affiches et images
  - Consultation des notes et critiques

  - exportation de films via l'api puis parsing des données pour en faire une base de données

### Application Pokémon

- Consultation des données :
      - Recherche de Pokémon par nom ou numéro
      - Affichage des statistiques détaillées
      - Visualisation des sprites et artworks
      - Information sur les types et capacités

- Système de combat :
      - Sélection de deux Pokémon
      - Comparaison des statistiques
      - Simulation de combat
      - Calcul des dégâts et efficacité


## Technologies Utilisées

  - Python 3.9+
  - SQLite pour la base de données
  - Streamlit
  - Pandas
  - TMDB API (The Movie Database)
  - PokéAPI

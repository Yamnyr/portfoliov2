## Description

HurleaV2 est une plateforme de streaming, développée sous Symfony. Met l'accent sur l'expérience utilisateur avec une interface modernisée et de nouvelles fonctionnalités comme l'historique de visionnage, un système de favoris amélioré et un catalogue enrichi avec des options de filtrage avancées.

## Fonctionnalités

- Système d'authentification sécurisé
- Profils personnalisables avec avatars
- Historique des activités et connexions
- Interface de navigation fluide et responsive
- Filtres avancés :
     - Par genre
     - Par année
     - Par note
     - Par durée
     - Par pays
     - Par acteurs/réalisateurs
- Système de tri multiple :
     - Par popularité
     - Par date d'ajout
     - Par note moyenne
     - Par ordre alphabétique
- Player vidéo permettant de regarder des films et/ou séries
- Système de favoris
- Historique

### Administration

- Gestion complète du catalogue
- Monitoring des utilisateurs
- Statistiques d'utilisation :
 - Nombre de visionnages
 - Temps de visionnage moyen
 - Contenus les plus populaires
- Modération des commentaires
- Gestion des droits d'accès
- gestion de l'espace de stockage sur lequel est uplodé les films

## Technologies Utilisées

### Backend
- Symfony 6.4
- PHP 8.1+
- MySQL
- Doctrine ORM

### Frontend
- Twig
- JavaScript
- SASS
